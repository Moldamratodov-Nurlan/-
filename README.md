# 📝 Мәтін анализаторы (Text Analyzer)

## 🎯 Мақсаты:
Бұл жобаның мақсаты — String және коллекциялармен жұмыс істеуді үйрену. Бағдарлама мәтінді талдап, сөздер санын және әр сөздің жиілігін анықтайды.

---

## 📌 Жобаның сипаттамасы:
Бағдарлама енгізілген мәтінді өңдейді. Мәтінді сөздерге бөліп, олардың санын есептейді және әрбір сөздің неше рет кездесетінін анықтайды. Бұл үшін HashMap коллекциясы қолданылады.

---

## ❓ Сұрақтар және жауаптар:

### 1. split() қалай жұмыс істейді?
split() әдісі мәтінді белгілі бір бөлгіш (delimiter) арқылы бөліктерге (сөздерге) ажыратады. Нәтижесінде массив (array) қайтарылады. Мысалы, бос орын бойынша бөлсе, мәтін сөздерге бөлінеді.

---

### 2. HashMap не үшін керек?
HashMap — кілт-мән (key-value) форматында деректерді сақтайтын құрылым. Бұл жерде:
- кілт (key) → сөз  
- мән (value) → сол сөздің саны  
HashMap әр сөздің жиілігін тез есептеуге мүмкіндік береді.

---

### 3. Жиілік қалай есептеледі?
Жиілік әр сөздің неше рет кездескенін санау арқылы есептеледі:
- Егер сөз бірінші рет кездессе → 1 деп сақталады  
- Егер бұрын бар болса → мәні 1-ге артады  

---

## 📊 Қорытынды:
Бұл жоба мәтінмен жұмыс істеу дағдыларын дамытады. String әдістері мен коллекциялар (әсіресе HashMap) арқылы деректерді тиімді өңдеуге болады. Нәтижесінде мәтін бойынша толық статистика алынады.
